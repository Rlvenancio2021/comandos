## Comando git para filtro
 - git filter-branch --force --prune-empty --index-filter 'git rm -rf --cached --ignore-unmatch password.txt' --tag-nome-filter cat -- --all
 - Onde 'password.txt' é o nome do arquvio neste exemplo

## Comando Terminal OS
 - pwd - mostra o caminho da pasta atual
 - cd - abre o diretorio
 - cd ~/<nome do diretorio> abre o diretorio
 - cd ../<nome do diretorio> troca de um diretorio para o outro em um único comando
	//Caso necessário voltar duas pastas fica "cd ../../<caminho do diretório desejado>"
 - command + k - Limpa a tela do terminal
 - command + l - Limpa parcialmente a tela do termianl
 - clear - Limpa a tela do terminal
 - mkdir <nome do diretorio> - Comando para criar um diretorio
 - rm -rf - comamdo para apagar diretorio e/ou arquivos
 - ls - Comando para verificar conteúdo do diretorio
 	- Variações: ls -a - mostra arquivos ocultos
		     ls -l - mostra os arquivos de forma mais estruturada
		     ls -al ~/ - mostra todas as pastas a partir da raiz
